# Monke-Modding-Maker
Gorilla tag tools to make gorilla tag mods!

# What you get
1. Mod manager
2. Simple maps to use as a base
3. Modding tools
4. Unity Plugin


# Requirments
1. Node.js (https://nodejs.org/en/)
2. Unity (https://unity.com/)
3. Basic knowledge of modding/ C# coding (tutorialspoint.com/csharp/index.htm)
